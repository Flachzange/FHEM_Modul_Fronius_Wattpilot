# Historical API compilation provenance

The former root `API.md` was a broad historical field compilation. It was replaced as an active reference because its aliases, meanings, units, enum values, types, categories, examples, and read/write claims were not consistently verified for Wattpilot Flex.

Its useful operational descriptions were transferred, with explicit evidence qualification, to [`WATTPILOT-FLEX-FIELD-DESCRIPTIONS.md`](WATTPILOT-FLEX-FIELD-DESCRIPTIONS.md). The Git history before Issue #25 remains the archival source for the complete former table.

Credential-like examples, private endpoint patterns, network identifiers, and unverified write claims were deliberately not republished into the active companion catalog. The complete sanitized Flex key set and JSON shape remain in [`WATTPILOT-FLEX-JSON-API.md`](WATTPILOT-FLEX-JSON-API.md).
